# restaurant_customer_socket


Backend :

cd backend
npm install
npm start

    tests:
    curl http://localhost:3000/health
    curl http://localhost:3000/api/orders

Frontend :

cd frontend
php -S localhost:8089

    Customer View :
    http://localhost:8089/

    Admin View :
    http://localhost:8089/admin.html
